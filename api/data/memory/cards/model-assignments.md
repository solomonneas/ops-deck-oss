## Model Roles
- Fast models handle triage, classification, and short summaries.
- Balanced models support drafting, planning, and code review.
- Deep reasoning models are reserved for high-complexity investigations.

## Decision Rule
Choose the lightest model that can complete the task reliably, then escalate only when quality drops.
